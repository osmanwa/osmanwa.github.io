# Personal Website

A professional, classic personal website built with HTML, CSS, and JavaScript.

## Features

- Clean, modern design with classic typography
- Fully responsive layout
- Smooth scrolling navigation
- GitHub Pages deployment ready

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the `master` branch.

To enable GitHub Pages:
1. Go to your repository Settings
2. Navigate to Pages
3. Under "Source", select "GitHub Actions"

The workflow will automatically build and deploy your site.

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then visit `http://localhost:8000` in your browser.
